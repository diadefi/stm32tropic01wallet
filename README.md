# STM32 + TROPIC01 Bitcoin Hardware Wallet Prototype

Open-source Bitcoin hardware-wallet prototype using STM32U5 firmware, TROPIC01 secure-element authentication, UART commands, and PowerShell hardware regressions.

## Features

- Device-side CHECK/SIGN policy engine
- CHECK_ID transaction commitments
- PIN/session unlock model
- TROPIC01 authentication gate
- Physical/user confirmation path
- Bitcoin Core regtest signing loop
- Testnet-readiness milestones
- PowerShell regression harness
